### Позитивные тесты

1. 1 2 - 0.25
2. Все числа одинаковые - 0
3. 10 случайных чисел, дисперсия с [сайта](https://calculator-online.net/ru/variance-calculator/) - 12.49
4. 25 случайных чисел - 40.48
5. 50 случайных чисел - 40.93
6. 500 случайных чисел - 36.36
